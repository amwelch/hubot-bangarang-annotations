# hubot-bangarang-annotations

## Installation

In hubot project repo, run:

`npm install hubot-bangarang-annotations --save`

Then add **hubot-bangarang-annotations** to your `external-scripts.json`:

```json
[
  "hubot-bangarang-annotations"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
